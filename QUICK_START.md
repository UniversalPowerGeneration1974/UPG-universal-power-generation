# Quick Start Guide for UPG Application

## Windows Users - Quick Start

### Backend Setup (PowerShell or CMD)

```powershell
# 1. Navigate to backend folder
cd backend

# 2. Install dependencies
npm install

# 3. Start the server
npm start
```
Backend will run at: **http://localhost:5000**

### Frontend Setup (in new terminal)

```powershell
# 1. Navigate to frontend folder
cd frontend

# 2. Install dependencies
npm install

# 3. Start React development server
npm start
```
Frontend will run at: **http://localhost:3000**

## Testing the Application

1. **Create Account**
   - Click "Sign Up"
   - Fill in: Full Name, Username, Email, Password
   - Click "Sign Up"

2. **Create Another Account** (for testing chat)
   - Open new browser/incognito window
   - Repeat signup process with different credentials

3. **Test Chat**
   - Go to "Chat" page
   - Select a contact from the list
   - Send messages

4. **Update Profile**
   - Go to "Profile" page
   - Edit your information
   - Click "Update Profile"

## Verify Everything Works

- ✅ Can sign up with new account
